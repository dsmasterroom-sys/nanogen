# Nanogen 프로젝트 작업 지시사항 (User Instructions)

이 파일은 **Nanogen 프로젝트**를 함께 작업할 때 AI 에이전트가 **반드시 준수해야 하는 규칙과 지침**을 기록하는 곳입니다.
작업을 시작하기 전에 이 내용을 확인하고 진행해 주세요.

---

## 🚩 기본 원칙
1. **언어**: 모든 의사소통과 계획 문서는 **한국어**로 작성합니다. (UI 텍스트는 원본 유지를 위해 영어 사용 가능)
2. **코드 스타일**:
   - Python: PEP 8 준수
   - JS/HTML: 가독성을 위해 들여쓰기 유지
3. **경로**: 프로젝트 루트는 `c:\Users\DS\Master\nanogen_django` 입니다.

## 🛠️ 기능 구현 지침
- [여기에 지시사항을 추가해 주세요]
- 예: 새로운 모달을 추가할 때는 기존 스타일(Tailwind CSS)을 엄격히 따른다.
- 예: API 호출 로직 변경 시 `services.py`와 `views.py`를 모두 검토한다.

## ⚠️ 주의 사항
- `settings.py`의 `DATA_UPLOAD_MAX_MEMORY_SIZE`는 10MB 이상으로 유지해야 합니다.
- 프론트엔드 변경 시 `SyntaxError`에 주의하고, 괄호 짝이 맞는지 항상 확인해야 합니다.
